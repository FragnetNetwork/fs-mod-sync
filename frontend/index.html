FS Mod Sync

Sync mods from your Farming Simulator server

Step 1: Enable Public Mod Download

Before using this tool, you need to enable Public Mod Download on your Farming Simulator dedicated server control panel.

  1. Log in to your server control panel
  2. Navigate to the Settings tab
  3. Enable "Public Mod Download"
  4. Copy the mods link shown below it (looks like: http://IP:PORT/mods.html)
